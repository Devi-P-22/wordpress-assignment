# WordPress Assignment

## Overview
This project recreates the mobile homepage using WordPress and ACF.

## Features
- Dynamic content using ACF
- Editable hero section, brands, and products
- Mobile responsive design
- Real-time product search (JavaScript)

## Tech Used
- WordPress
- ACF Plugin
- HTML/CSS
- JavaScript

## Setup
1. Place theme inside wp-content/themes/
2. Activate theme
3. Install ACF plugin
4. Import/create fields
